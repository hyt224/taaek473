Coding - 당신의 계절은? 계절 성격 테스트

본인의 성격을 계절에 빗대어 알 수 있는 성격 테스트 프로그램입니다.
질문에 답하여 본인은 어느 계절에 속하는지 알아 보세요.

다운로드

1) 압축을 해제하고 seasonpersonalitytest 파일을 실행해 주세요. 보안상 열리지 않는다면 단축 메뉴를 통해 실행해 주세요.
2) start 버튼을 누르고 이어지는 16개의 질문에 답해 주세요.


3) 마지막으로 결과는? 버튼을 누르면 8개의 성격 중 본인에게 맞는 성격과 그에 대한 설명이 나옵니다.

4) 다시하기 버튼을 누르면 처음부터 테스트가 진행됩니다.

지인과 같이 해 보고 결과를 공유해 보아도 좋을 것 같습니다. 즐거운 시간 되세요!

해당 프로그램은 2021년 진행된 성균관대학교 러닝페어 '프로공감러' 팀의 제작물임을 밝힙니다.
